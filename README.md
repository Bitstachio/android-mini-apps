# Android Sandbox

A collection of small Android Studio projects exploring common app features and UI components.

## Brief Project Overviews

### [Contact Manager](https://github.com/Bitstachio/android-sandbox/tree/main/contact-manager)
A **contacts app clone** that mimics basic functionality of managing a list of contacts.
Covers creating, reading, updating, and deleting entries while practicing data handling and UI binding.

### [Player Roster](https://github.com/Bitstachio/android-sandbox/tree/main/player-roster)
A simple app demonstrating the use of **RecyclerView** for displaying a list of players.  
Users can browse through a roster, tap on an item to view **player details**, and learn about list navigation and adapters.

### [Profile Manager](https://github.com/Bitstachio/android-sandbox/tree/main/profile-manager)
A **profile details page** that allows users to view and update their personal information.  
Includes functionality for **picking and setting a profile image** from the device gallery, demonstrating the use of intents and permissions.

### [User Auth](https://github.com/Bitstachio/android-sandbox/tree/main/user-auth)
A simple **user authentication** demo with **login and registration** screens.  
Focuses on form validation, navigation between screens, and basic data persistence.
